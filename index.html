📨 Automated Message Cadence — Status Triggered
🟢
Price Drop Alert — sent to: Contacted, ShowingDesired, StillDeciding, AppSent, Ghosting, MissedShowing, ShowingFailed and a few more. Skipped if no inbound activity in 60 days.
🔴
Unit No Longer Available — sent to most active statuses. Skipped for: Withdrawn, NotInterested, Ghosting, ApplicationReceived+, LeaseSigned, MovedIn.
🔵
Unit Ready for Showing (back on market) — sent to: Contacted, New, ShowingDesired, ShowingCanceled only. All other statuses skipped.
⚫
Post-Showing Follow-up — blocked entirely if prospect is Withdrawn, Ghosting, or NotInterested at time of ShowingComplete.
🤖
These messages are not AI — they're status-triggered templates. Go to Settings › Communication › Messages to review and customize.
📬 Follow-Up Cadences by Stage
1️⃣
Pre-showing (New / Contacted / ShowingDesired) — up to 3 SMS nudges: ~24h → 20h → 24h → final 48h. If no reply → marked Unresponsive (Ghosting), follow-ups stop.
2️⃣
Incomplete Application — up to 4 SMS over ~3 days: +6h → +18h → +24h → +24h. Messages are personalized to the section they dropped off on. Final: "I'll go ahead and close out your file."
3️⃣
Post-Showing — App Link Sent — up to 3 SMS. Auto-send: first ~4h after showing, then +24h, +48h → Ghosting. Manual-send: first ~24h after agent sends, then same chain.
4️⃣
Post-Showing — Still Deciding — up to 3 SMS: ~18h after showing → +48h → +40h → Ghosting.
5️⃣
Failed / Missed / Canceled Showing — 1–2 reschedule nudges: immediate link → +24h if no rebook → stops.
🔔
Showing Reminders — separate from follow-ups. Accompanied: confirmation SMS 180min before, auto-cancel 90min before if unconfirmed. Self-guided: link 1h before, never auto-canceled. All reminders clamped to 8am–8pm local time.
🛑
Follow-ups stop immediately when status hits: NotInterested, Withdrawn, Ghosting, Blocklist, AppRejected, AppApproved, LeaseSigned, MovedIn, DuplicateLead, UnitUnavailable. Also pause if prospect replied within last 32h or unit is On Hold.

🤖 SMS AI — What It Handles vs. Skips

Module 3 — Command Center · (5/5)

Self-Guided Showing Flow

🔑 Self-Guided Prospect Experience
1️⃣
Ready for your tour? — Prospect clicks their link. Location check + timing gate (available 45min before, up to 60min after scheduled time). Must click "I've Arrived".
2️⃣
Access Instructions — Shows lockbox location, codes, gate codes, PM notes. Can tap "I'm in" or "I'm having trouble."
3️⃣
Feedback form — Required: next step (apply / still deciding / not interested). Optional: likes, dislikes, star rating. Must lock up before submitting.
⏱️
Auto-complete: 45 min after scheduled time, if status is ShowingStarted/Arrived → marked Complete → prospect gets feedback SMS + application link. If they never arrived → marked MissedShowing.
❌
Self-guided showings are NOT auto-canceled — no confirmation required from the prospect. They stay in ShowingScheduled indefinitely until they arrive or 45min passes.
